Type annotations

Só pra lembrar...

>>> def(name: str) -> str:
...     return 'Hello ' + name

Não signfica que o CPython valida o parâmetro ou o retorno. 

>>> captain: str    # None
>>> values: List[int] = [1, 2, 4]

Essas informações podem ser extraídas de __annotations__ (assim como docstrings podem ser extraídas de __doc__).
